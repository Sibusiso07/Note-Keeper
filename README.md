Note Keeper

Note Keeper is a web application created as part of Dr. Angela Yu's Web Development course on Udemy. The application is built using React and provides users with the ability to add and delete notes effortlessly.

Features

User-Friendly Interface: The application offers a clean and intuitive user interface, making it easy for users to interact with their notes.

React Framework: Note Keeper is developed using React, a powerful JavaScript library for building user interfaces. This ensures a fast and responsive experience for users.

Note Management: Users can seamlessly add new notes to the application. Additionally, the delete functionality allows users to remove notes they no longer need.
